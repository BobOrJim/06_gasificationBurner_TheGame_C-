1. dubelklicka på klicka på SFC_totalBadass

2. se bild 1, ställ in upplösningen. Pga alla beräkningar är spelet gjort i 720 (HD-ready). Om du planerar spelkväll ändra din skärms upplösning för en betydligt bättre spelupleelse

3. håll in CTRL och klicka som bild 2 visar, välj sparad fil 66a. Denna sparade fil är byggd ihop med programmet.

4. Välj electronics och sedan spark som bild 3 visar. Spark används för att styra saker. Gula objekt går nu att klicka på

5. Tryck space för att starta spelet, och för att pausa igen, mkt användbart när man inte hänger med.

6. När du förstört pannan gå till punkt 3.


PS: En bug är känd, om pannan exploderar kan den inte explodera igen, utan att programmet stängs av och sedan startas om igen. Är man en badass så startar man således om spelet igen.
